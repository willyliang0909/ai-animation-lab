# 預期流程

1. 請ai閱讀圖片並整理出關鍵畫面
2. 確認畫面敘述後請ai協助提供生成畫面所需要的提示詞
3. 角色定裝及語音提示詞
4. 生成角色
5. 生成場景
6. 將角色帶入場景及劇情中生成影片
7. 生成對話語音
8. 生成背景音樂
