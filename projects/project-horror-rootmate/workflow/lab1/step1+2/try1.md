# 請ai閱讀圖片並整理出關鍵畫面

😊
```
你是一個專業的小說改編ai動畫家，就當前內容改編
應該給出幾個關鍵畫面
注意這是一則靈異故事，以第一人稱敘事為主，配合重現畫面的對話
```

## note
> 只讀照片沒問題，且可以顧到故事順序，場景動作安排也超乎預期
## 問題
> 心中認為最關鍵的點沒被安排進去

```
這是一個非常吸引人的開端！根據你提供的這幾頁《十大猛鬼凶屋》內容，我為你規劃了四個關鍵的動畫畫面，完美契合靈異、第一人稱敘事以及重現對話的風格。

這段故事描繪了一個從國外回港的女孩，因為預算限制，陰錯陽差地住進了一間唐樓，並與一位名叫「韻如」的神秘女生合租。故事在平凡中透出詭異，特別是韻如第一次見面時說的那句「我等你好久了」，為後續的靈異發展埋下了深刻的伏筆。

以下是四個關鍵畫面的詳細規劃：

畫面一：香港的街頭陰影
【場景說明】 雖然是白天，但香港老街區的窄巷裡卻顯得昏暗。鏡頭以第一人稱視角（主角 POV）掃視。街上有些擁擠，霓虹燈牌招牌（即使白天也亮著）在老舊建築間交錯。

【視覺風格】 畫面呈現出一種壓抑的灰調，與主角剛回國的興奮感形成強烈對比。背景聲嘈雜，但被一層低頻的嗡嗡聲蓋過。

【關鍵動作/細節】 鏡頭停在一根貼滿各種招牌的電線桿上。主角的手（從鏡頭外伸出）指向其中一張手寫的、略微泛黃的合租廣告。

【旁白/對話】

主角（旁白，略帶疲憊）：來到香港的頭幾天，住飯店太貴了。我在街上閒逛，想找個便宜的地方落腳。然後，我看到了這個……

（鏡頭聚焦在廣告上，上面寫著：近港鐵站、交通方便、誠招女伴合租、請電：9425 XXXX）

畫面二：幽暗唐樓的初見
【場景說明】 場景轉移到唐樓（Old Hong Kong Tong Lau）的樓梯間。這裡極度昏暗，只有牆上高處一個小小的氣窗透進來一點點慘白的光。牆壁斑駁脫落。

【視覺風格】 畫面變得更冷、更暗。採用窄角鏡頭，強調空間的狹窄和壓迫感。空氣中似乎有微塵在光柱中飄散。

【關鍵動作/細節】 主角和另一個女生（韻如）正並肩走上樓梯。韻如走在前面，身形瘦削，穿著簡單。她沒有回頭，主角只能看到她的背影。

【旁白/對話】

主角（旁白，緊張）：她約我在單位附近的公園見面。她叫韻如，一個很清秀的女孩，沒化妝，眉清目秀。

（重現對話 - 主角聲音略微顫抖）：我：「你好，請問是妳找人合租嗎？」

韻如（背對著鏡頭，聲音細細的，帶著一種古怪的柔和）：「是的。我等妳好久了。」

（主角 POV 鏡頭猛地一頓，強調這句話帶來的震驚）

畫面三：六樓的致命高度
【場景說明】 鏡頭依然是主角 POV，但此時她正氣喘吁吁地爬著永無止境的樓梯。這是一棟沒有電梯的六層唐樓。

【視覺風格】 畫面開始輕微晃動，模擬主角的呼吸和疲勞。樓梯間顯得越來越深不見底。

【關鍵動作/細節】 韻如依然輕鬆地走在前面，彷彿沒有重量。主角的手扶在生鏽的欄杆上，鏡頭轉向樓梯井中間，向下望去，是一個讓人眩暈的深淵。

【旁白/對話】

主角（旁白，氣喘吁吁）：六樓。對於平時沒做運動的我，真的很吃力。

（韻如的聲音從前方飄來，不帶任何情緒）：「妳什麼時候可以上來看看？」

畫面四：單位內的神秘室友
【場景說明】 進到單位內部。這是一個典型的唐樓間隔，雖然小但很通爽，光線比樓梯間好一些。韻如站在房間中央，光線從她身後的窗戶透進來，使她的臉處於陰影中。

【視覺風格】 畫面雖然亮了，但韻如的臉依然模糊不清。房間雖然空曠，卻給人一種莫名的寒意。

【關鍵動作/細節】 鏡頭掃視房間，韻如靜靜地站著，看著主角。主角的手再次出現，在租約上簽字。

【旁白/對話】

主角（旁白）：地方小小的，但很通爽。我心想，為什麼會說等我很久？

（聲音轉為主角的低語）：但我還是決定合租。

韻如（依然站在陰影中，聲音幽幽地）：「我是個大學生，想找女生合租減輕負擔。」

（畫面在主角簽字的一瞬間突然定格，並迅速轉為黑白，預示著噩夢的開始。）
```
